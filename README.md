# 東洋医学 弁証支援アプリ

東洋医学の**弁証鑑別・治療穴の提示・徒手検査フロー・カルテ管理**を1つのHTMLファイルで行う、学習・臨床補助ツールです。インストール不要・通信不要で、ブラウザだけで動作します。

> ⚠️ 本アプリは学習および臨床補助のための**鑑別支援ツール**です。表示される弁証・治療穴・検査は候補であり、確定診断ではありません。最終的な判断は施術者ご自身が行ってください。

## 主な機能

- **問診** — 患者番号・氏名・来院日・予約/来院時間・主訴を記録（患者番号は全角/半角を区別せず照合）
- **症状入力** — 約60項目のチェックリスト
- **徒手検査** — 部位ごとの整形外科テストと判定フロー（陽性所見をチェック）
- **弁証** — 八綱・気血津液・臓腑・経絡弁証を統合し、合致度の高い証と治療穴（補瀉つき）を提示
- **治療内容／施術後・次回予定** — 実施した処置と経過を記録
- **カルテ管理** — 患者番号・氏名で検索、来院日順に時系列表示、記録の編集（編集履歴つき）
- **入出力** — JSONバックアップ／読込（マージ対応）、CSV出力、患者配布用の問診表を印刷/PDF出力
- **患者番号 ⇄ 氏名 の自動補完**

## 公開（GitHub Pages）

1. このリポジトリの **Settings → Pages** を開く
2. **Build and deployment** の **Source** を「Deploy from a branch」にする
3. Branch を `main`（または `master`）/ フォルダを `/ (root)` に設定して保存
4. 数十秒〜数分で `https://<ユーザー名>.github.io/<リポジトリ名>/` に公開されます

`index.html` がトップに表示されます。

## データの保存について

- カルテは**開いたブラウザの localStorage** に保存されます（外部送信は一切ありません）。
- 保存はブラウザ単位です。別のブラウザ・別の端末とは共有されません。
- ブラウザの「サイトデータ削除」を行うとカルテも消えます。
- **GitHub Pages で公開した URL から使うと、正式なドメインを持つため localStorage が安定して保持されます**（ローカルファイル `file://` での Safari 7日自動削除などの影響を受けにくくなります）。
- 大切なデータは、アプリ内の **「⬇ バックアップ(JSON)」** で定期的に書き出して保管してください。別端末への移行や復元は **「⬆ 読込(JSON)」** で行えます。

## ローカルでの利用

`index.html` をブラウザで開くだけで動作します。
